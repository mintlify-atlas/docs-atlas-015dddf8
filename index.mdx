## Why ADK Utils?

Google's Agent Development Kit (ADK) provides excellent support for local models in Python via `litellm`, but **there is no native way to use local Ollama models with TypeScript**. ADK Utils bridges this gap with a complete toolkit for building TypeScript agents.

## Quick Start

Get started with ADK Utils in three simple steps:

<Steps>
  <Step title="Install the package">
    <CodeGroup>
    ```bash npm
    npm install @yagolopez/adk-utils
    ```
    ```bash yarn
    yarn add @yagolopez/adk-utils
    ```
    ```bash pnpm
    pnpm add @yagolopez/adk-utils
    ```
    </CodeGroup>
  </Step>
  
  <Step title="Install peer dependencies">
    ```bash
    npm install @google/adk @google/genai ai ollama
    ```
  </Step>
  
  <Step title="Create your first agent">
    ```typescript
    import { OllamaModel } from '@yagolopez/adk-utils';
    import { LlmAgent } from '@google/adk';

    const agent = new LlmAgent({
      name: 'LocalAgent',
      model: new OllamaModel('qwen3:0.6b', 'http://localhost:11434/v1'),
      description: 'Agent description',
      instruction: 'You are a helpful assistant...',
    });
    ```
  </Step>
</Steps>
